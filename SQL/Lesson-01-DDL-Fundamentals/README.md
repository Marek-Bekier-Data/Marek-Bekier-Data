# 📊 Lesson 1: DDL, DML, DQL Fundamentals & Data Import

First steps in working with relational databases. I focused on defining database structures and preparing data for analysis.

### 🛠️ What I learned:
*   **Table Creation (`CREATE TABLE`)**: Defining column names and choosing appropriate data types.
*   **Data Types**: 
    *   `VARCHAR` / `CHAR` – for storing text (e.g., names, cities).
    *   `INTEGER` – for whole numbers (e.g., age, IDs).
*   **Data Management**: 
    *   `UPDATE` – modifying existing records.
    *   `DROP TABLE` – safely removing unnecessary structures.
*   **ETL Process**: Importing raw **CSV** files into the database using the **SQLiteOnline** tool.
